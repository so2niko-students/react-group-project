## Branch naming convention
- **prefix A**
  - feat - new feature
  - fix - fixing something
  - docs - adding documentation/rewriting
- **prefix B**
  - OT - Olena
  - VP - Viktor
  - MS - Mykola
- **ticket_number**
- **ticket_name**

**Result**: 
- A/B_ticket_number_ticket_name
- *feat/OT_2_init_project*
- *fix/VP_33_fix_cors_etc*
